# GenTube

Transform your selfies into stunning AI art in seconds. Create professional headshots, viral memes, or artistic portraits with AI-powered transformations.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for production
```bash
npm run build
```
